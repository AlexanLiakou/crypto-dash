
# Crypto Overview

A simple crypto prices listing application. It consumes the [CoinGecko](https://www.coingecko.com/en/api) API.

<br><br>

## Tech Stack

**Main:** [React.js](https://react.dev/), [Vite](https://vite.dev/)

**Packages:** [Chart.js](https://www.chartjs.org/), [React Spinners](https://www.npmjs.com/package/react-spinners)


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`VITE_API_URL='https://api.coingecko.com/api/v3/coins/markets?vs_currency=eur'`

`VITE_API_COIN_URL='https://api.coingecko.com/api/v3/coins'`

## Run Locally

Clone the project

```bash
  git clone https://github.com/AlexanLiakou/crypto-dash.git
```

Go to the project directory

```bash
  cd crypto-overview
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
